# 学后复盘模板

遵循 `learning-review` skill 模式 A，在学习笔记完成后立即复盘。

## 目录

确保目录存在：`mkdir -p <workspace>/learning/reviews/post-learning`

## 模板

写复盘报告到 `<workspace>/learning/reviews/post-learning/YYYY-MM-DD.md`：

```markdown
# 学后复盘 - YYYY-MM-DD

## 学习主题
<topic>

## 一句话总结
<费曼法：用最通俗的话概括核心>

## 和我工作的关系
- 直接能用：<列出可以马上应用的点，具体到"在什么场景下做什么">
- 未来有用：<需要积累或特定条件才用得上的>
- 无关：<学完发现跟自己领域不大的>

## 待内化项
- [ ] → AGENTS.md：<需要更新的具体内容>
- [ ] → TOOLS.md：<需要更新的具体内容>
- [ ] → SOUL.md：<需要更新的具体内容>
- [ ] → MEMORY.md：<需要记录的重要洞察>

## 遗留问题
<学完还搞不懂的，或需要进一步探索的>
```

## 规则

- 没有就写"暂无"，不硬凑
- 不推群，安静执行
- 复盘在学习笔记完成后立即进行
