# 四维度评分标准详解

> 从 SKILL.md 拆出的详细评分标准，按需加载。

## 维度1：财务质量（30%）

| 指标 | 阈值 | 满分 |
|------|------|------|
| ROE | ≥15% | 30分 |
| 毛利率 | ≥30% | 25分 |
| 经营现金流/净利润 | ≥1.0 | 25分 |
| ROA | ≥8% | 20分 |

| 等级 | 分数 | 条件 |
|------|------|------|
| 优秀 | 90-100 | 所有指标达标 |
| 良好 | 75-89 | 3/4指标达标 |
| 合格 | 60-74 | 2/4指标达标 |
| 较差 | <60 | 少于2个达标 |

## 维度2：估值（30%）

| 指标 | 阈值 | 满分 |
|------|------|------|
| PE-TTM | ≤30 | 40分 |
| PB | ≤5 | 30分 |
| PEG | ≤1.5 | 30分 |

| 等级 | 分数 | 条件 |
|------|------|------|
| 优秀 | 90-100 | PE≤15且PB≤2 |
| 良好 | 75-89 | PE≤30且PB≤5 |
| 合格 | 60-74 | PE≤50或PB≤8 |
| 较差 | <60 | PE>50且PB>8 |

## 维度3：护城河（25%）

### 护城河类型
- **品牌护城河**：知名品牌、消费者忠诚度
- **成本优势**：规模效应、原材料掌控
- **网络效应**：用户越多价值越大
- **转换成本**：客户粘性高
- **专利壁垒**：核心技术优势

| 等级 | 分数 | 条件 |
|------|------|------|
| 强护城河 | 90-100 | 2种以上 + 数据支撑 |
| 中等护城河 | 75-89 | 1-2种 + 有证据 |
| 弱护城河 | 60-74 | 有护城河但证据不足 |
| 无护城河 | <60 | 无明显护城河 |

## 维度4：管理层（15%）

| 指标 | 阈值 | 满分 |
|------|------|------|
| 分红率 | ≥30% | 40分 |
| 透明度 | 高/中/低 | 30分 |
| 战略执行 | 优秀/良/中 | 30分 |

| 等级 | 分数 | 条件 |
|------|------|------|
| 优秀 | 90-100 | 高分红率 + 高透明度 |
| 良好 | 75-89 | 中等分红 + 透明度尚可 |
| 合格 | 60-74 | 低分红但透明 |
| 较差 | <60 | 低分红 + 低透明度 |

## 综合评分计算

```
综合分 = 财务质量分 × 0.30 + 估値分 × 0.30 + 护城河分 × 0.25 + 管理层分 × 0.15
```

四舍五入到一位小数。
